## 数据库表结构

### 反馈表（feedbacks）
- id, category, encrypted_content, device_id, created_at
- status: pending/in_progress/resolved/no_solution
- solution, solution_updated_at, solution_admin_ip

### 评论表（feedback_comments）
- id, feedback_id, commenter_type, content, created_at, admin_ip

### 访问日志表（access_logs）
- id, ip, user_agent, path, method, status_code
- is_bot, is_suspicious, suspicious_reason, response_time
- created_at

### 访问统计表（access_stats）
- date, total_visits, unique_visitors, bot_visits
- suspicious_visits, avg_response_time

### IP 黑名单表（ip_blacklist）
- id, ip, reason, attempts, blacklisted_at, expires_at

### 管理员日志表（admin_logs）
- id, action, target_id, admin_ip, created_at

## 技术栈

- Express 5.2.1

- express-rate-limit (速率限制)
