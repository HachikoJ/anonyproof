## 📈 数据说明

### 访问统计
- **今日访问** - 今天所有请求的数量
- **独立访客** - 今天不同的 IP 数量
- **爬虫访问** - 检测到的爬虫请求数
- **可疑访问** - 检测到的可疑访问数
- **黑名单 IP** - 当前在黑名单中的 IP 数量

### 访问日志
- **时间** - 访问时间
- **IP** - 访问者 IP
- **方法** - HTTP 方法（GET/POST 等）
- **路径** - 请求的路径
- **状态** - HTTP 状态码
- **响应时间** - 服务器响应时间
- **标签** - 爬虫/可疑标记

### IP 黑名单
- **IP** - 黑名单 IP 地址
- **原因** - 加入黑名单的原因
- **违规次数** - 触发速率限制的次数
- **加入时间** - 被加入黑名单的时间
- **过期时间** - 黑名单过期时间（默认 24 小时）

---

## 🔐 安全提示

### 黑名单规则
- 1 小时内触发速率限制 10 次自动加入黑名单
- 黑名单默认有效期 24 小时
- 黑名单 IP 无法访问任何页面（返回 403）

### 检测规则
- 空 User-Agent
- 脚本特征（curl、wget、python 等）
- 敏感路径访问（/admin、/.env 等）
- 异常访问模式
