Product-grade anonymous feedback system

匿名反馈，也可以做得像一款成熟产品。

AnonyProof 把匿名提交、隐私保护、通知中心、状态跟进与双向沟通整合进一套轻量系统里，让反馈不再石沉大海。

适用于校园、企业、团队、社群或任何需要安全反馈机制的场景。

Core Features

不只是匿名，更是完整反馈闭环

从内容提交、状态变化到管理员回复与通知触达，整套体验围绕“反馈被认真处理”来设计。

🔒

完全匿名

无需注册登录，尽量减少额外身份暴露与使用门槛。

🛡️

端到端加密

在客户端完成加密处理，降低反馈内容暴露风险。

👁️

可审计操作

管理操作留痕，让后台处理过程更透明、更可追踪。

💬

双向沟通

支持评论与追加说明，让处理过程不止停留在单次提交。

🔔

通知中心

新消息与状态更新可快速直达，减少重复确认成本。

📱

多端适配

桌面与移动端都保持清晰、轻量且可用的交互体验。
