📸 界面展示

所有图片与说明均按已确认映射整理，顺序为管理端 → 用户端桌面版 → 用户端移动版

管理端界面 💻 桌面版

管理端主页

设备认证管理后台界面，显示总反馈 52 条（建议 37、投诉 6、举报 8），含分类筛选、状态筛选、搜索框、反馈列表、导出数据、退出按钮与通知图标。

管理端评论页

反馈界面显示“已有解决方案”与“讨论与沟通”区，展示用户与管理员对话记录，底部有回复输入框与“收起评论”按钮。

用户端界面 💻 桌面版

用户端带通知电脑版主页

左侧为“匿证”首页，包含“开始匿名反馈”“我的反馈”按钮与“端到端加密·完全匿名”提示；右侧为通知中心，显示未读通知与已读通知。

用户端我的反馈页电脑版

页面标题为“我的反馈”，展示设备 ID、反馈统计、搜索框、分类筛选、状态筛选与反馈列表，以及底部加密存储提示。

用户端界面 📱 移动版

用户端主页移动版

移动端首页展示“匿证 - 匿名反馈平台”、通知铃铛、总反馈数 53、加密率 100%、数据泄露 0，以及“端到端加密 / 防偷看机制 / 设备绑定”等功能说明。

用户端通知详情页移动版

通知中心显示未读通知（2）与已读通知（10），包含管理员回复、状态更新以及“标为已读”“查看详情”等操作入口。

用户端反馈详情页移动版

反馈详情页展示反馈类型、状态、提交内容、处理进展、评论区与管理员回复，完整体现反馈跟进流程。
